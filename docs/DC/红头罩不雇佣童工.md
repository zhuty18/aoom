# 红头罩不雇佣童工

当大堂经理大卫第一次对杰森提出这个建议时，杰森的心态变化过程大致是迷惑——震惊——恶心——唾弃——暴怒——抽出枪来在大卫脚前打了个洞。但冷静下来后他想了想，感觉好像也不能全怪大卫。毕竟大卫并不知道他和蝙蝠侠的关系，提议增设“蝙蝠侠cosplay钢管舞”也只是纯粹为了生意考虑。

自红头罩接手冰山赌场以来，他们停掉了之前绝大部分的业务，包括但不限于：走私、组织卖淫、器官贩卖、人口贩卖、雇佣杀人、高利贷等等等等。但随之而来的代价就是，单靠赌场这点生意，谁也没办法赚来足以运营一艘海上游轮的钱。杰森看着连月赤字的报表，陷入思虑：大概，是时候再去薅一波黑面具的羊毛了。

在哥谭黑吃黑这种事，杰森做得熟极而流，他轻而易举地扫荡了黑面具的三个仓库，因为夜生活的无聊而打了个哈欠，指挥小弟们给被绑起来的少女们一一松绑。最大的也不过十四五岁，用脚指头都猜得出黑面具是要干什么买卖。

军火、药物杰森毫不客气地笑纳去补充现金流了。另有些包含氪石在内的大规模杀伤性武器原材料，属于蝙蝠侠管控物品，他给提姆打了个电话，问他愿意出多少钱收购。他告诉提姆自己也会联系卢瑟。虽然他并没有去打卢瑟的电话，但提姆也只需要一个抬价的借口。

可如何处理这几十个小姑娘……杰森切切实实地犯了难，她们来自中美和南美，说的不是西语就是葡语，哥谭的孤儿院可不是语言学校，语言都不通的她们极有可能遭到二次拐卖，但要杰森解决——首先，他没那个钱做慈善；其次，要他雇佣童工？怎么说呢，他杰森·陶德又不是蝙蝠侠。

红头罩不雇佣童工。

因囚禁而营养不良的少女们面对黑洞洞的枪口，恐慌地聚在一团，杰森挑了一个看起来还算冷静的，走到她身边，半弯下腰用西语问道：“你们都会干什么？”

最终结果是，杰森在拉丁区多开了好几家餐馆、洗衣店等生意，甚至给每家店都配了保镖。

这些生意都很好，杰森这么告诉自己，都很方便洗钱。但他又实在忍不住心想：如果我有钱可以洗就好了。

虽然洗钱也是一门暴利营生，但是以黑吃黑著称的红头罩先生，实在很难谈下这类生意。

但杰森对科莉的看法不屑一顾：切，好人？好人能值几个钱？

堕落的第二步，是他开始出千了。提姆用黑市市价收购了他抢来的材料，为他提供了充足的启动资金，可以把赌场重新开起来。他开始让人在监控里定位那些人傻钱多的白痴，然后派人去把他们赢到只剩内裤为止。

堕落的旅途只走了两步，杰森就遇到了麻烦——布鲁斯·韦恩走进了冰山赌场的大门。

虽然不知道为啥老板会把他放到监视名单的第一个，但在目击他之后，大卫立刻上报了杰森。杰森差点从椅子上摔下来。无他，他只是正在面试钢管舞男。当然不是扮演蝙蝠侠的那种，杰森的羞耻心还没堕落到那个地步。

是扮演夜翼的。

杰森也说不清自己在想些什么，但他的第一反应确实是低头看了看那张卡——哥谭银行的金卡。如果布鲁斯过去六年没改过的话，每日免密支付额度是一万美元。

“咦，玛缇尔，餐厅的布局好像变了？”有一阵子未曾光顾的客人走进赌场，跟相熟的服务员闲聊。

玛缇尔把他带到预留的位置上：“是的。您来的很巧，今天有新节目第一次演出。我们给您留了最好的位置，希望您度过愉快的一夜。”

布鲁斯·韦恩兴致盎然地笑起来，坐下期待起这一场演出。

伴随着闪烁的灯光和干冰蒸腾出的雾气，一只小小的蝙蝠侠吊着威亚荡进舞台，翻了一个技术难度极高的跟头，模拟蝙蝠侠用抓钩枪在城市中游荡。

布鲁斯无声地睁大眼睛：达米安？

蝙蝠侠闪亮登场之后，自然少不了罗宾。这个罗宾从身形上就看得出早已不是小孩，但他花俏的跟头很好地弥补了这一点，马戏团空中飞人炫技的癖好让他看起来像当年一样充满童趣。——甚至可能更童趣一点，因为他当年的蝙蝠侠并不会像现在这个这样，压着还没进入变声器的嗓子让他低调点。

……更不会莫名其妙地跳起舞来。

布鲁斯闭上了眼睛——毋庸置疑这是一场精心筹备的演出，灯光、道具、音乐都配合得恰到好处，年幼的蝙蝠侠和成熟的罗宾更带来了一种令人捧腹的喜剧效果。——而这甚至不妨碍它把性暗示从钢管舞中完美剥离。

就像那只粉红毛毛兔一样，当蝙蝠侠和罗宾背后频繁出现六十年代的艺术字和声效来配合动作时，再饥渴的人也不得不承认，它是一场适合全年龄人观看的钢管舞。

<br>

“用不正规的价格提供正规服务。”杰森满意地看着上个月勉强实现收支平衡的财报，“我怎么早没想到这招呢？”

大卫看着奖金数额，也露出感动的笑容：“老板，你的商业顾问是个天才！蝙蝠侠和罗宾的演出这一个月来场场爆满，照这样下去，我们白道的盈利甚至可以超过黑道！”

杰森点点头：“他确实是个邪恶的小天才。”

大卫搓着手：“为什么咱们不把他正式招进来呢？从兔女郎到钢管舞，这样的人才，可不能被竞争对手抢走！”

“因为他未成年。”杰森·陶德冠冕堂皇地道，神色坦荡地好像莱克斯·卢瑟在说“我从不嫉妒超人”。

——“红头罩不雇佣童工。”

当然，更大的可能是，他开不起提姆·德雷克的薪水。
