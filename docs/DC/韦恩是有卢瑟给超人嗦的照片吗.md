# 韦恩是有卢瑟给超人嗦的照片吗

卢瑟宣布参选总统，是在几个月前发生的事。此后，他便和民主党、共和党的候选人一共角逐大选。两个对手一个素有恋童之名，另一个则沉迷煽动反智民粹，相比起来，卢瑟的声誉无疑要好得多，这也是他着力维护的一大竞争优势。

——直到那封邮件的曝光。

卢瑟不是世界上唯一一个有声誉的人。甚至，罪犯的名誉在许多时候比商人政客可靠得多。如果猫女说某毒贩保险柜里放了《共产党宣言》，那么大家都会相信，因为猫女确实可能看到所有人的保险箱。基于同样的道理，当一个交游广阔的皮条客说出一两句桃色故事，大家往往会倾向于相信，那就是真的。

即便那封邮件只是从一段漫长对话中摘下的短短一句，没头没尾，也足以震撼读者的认知。因为那臭名昭著的皮条客说道：

> 顺便问问他韦恩收到卢瑟嗦酥皮的照片了吗

<br>

有些理论天然难以证伪，盖因它能够切入某些频繁发生的困惑中，为其提供不经却合情合理的解释。我们不能因为自己目光狭隘就说它是错的——日心说曾被埋进故纸堆，地圆说曾被千夫所指，伽利略从斜塔上扔下两个球，人们的惊呼声震开绵亘千年俄迷雾，揭开了现代物理学的面纱。

因此，我们又有什么理由去先入为主，傲慢顽固地蔑视一种崭新的、合情合理的可能性呢？

凯特·格兰特写到这里，忍不住把脸埋进掌心，疯狂地笑了起来。

“凯特，你还好吗？”露易丝戳戳她不住抖动的肩膀。

凯特一下子趴在了桌上，边笑边说：“露，我有点绷不住了。”

露易丝沉默了一下，安慰道：“没事的，稿子发出去没人知道你没绷住。”

凯特：“可这是我待会儿要直播的新闻稿。”

露易丝捂住嘴，她也绷不住了。

她们身后，克拉克原本正三心二意写着《破产被收购后仍欠有十万债务：堪萨斯的悲歌》，此时也按捺不住，深吸一口气，离开了办公室。

他需要晒晒太阳。他从刷到那条邮件起就感到眼前一黑，他从不知道氪星人还可以低血糖，但话说回来，他也从没想到自己的性生活会成为全世界的网络顶流话题。虽然起源是英语，但其他语言的社交网络也没放过他。

他站在天台上冷静了一会儿，一度考虑今天要不要请假，但最终还是回到了办公室，继续写他的稿子。没办法，要吃饭的。小镇今年种大豆的邻居们损失惨重，克拉克想为他们做些什么，所以他真不太舍得那笔全勤奖。

克拉克回到办公室，屋里人声鼎沸，正在交换超人和卢瑟的梗图。超级视力让他在一瞬间受到了所有伤害，这个屏幕上把桌子后的超人配上“莱克斯·卢瑟与他的宿敌超人”的文字，那个屏幕上的维恩图以“给超人口过”作为肯特和卢瑟的共同点，佩里的屏幕上LexCorp的辟谣推送下被乐子人狂刷超人的名字。

甚至克拉克自己掏出手机一搜，第一条推送里都有纯白的液体从莱克斯嘴角溢出。

克拉克有种掉头就走的冲动，不是离开这件办公室，而是离开这个星球、这个星系、这个宇宙……甚至离开这个世界的念头都在他脑中一闪而过。

是手机铃声打断了他。

他抱着电脑躲进杂物间：“喂，布鲁斯。”

布鲁斯一如既往地言简意赅：“你看新闻了吗？”

克拉克眼前又是一黑：“……我倒是希望自己没看到。”

布鲁斯顿了一下：“平民的手机随时可能被遥控爆炸，你不在乎？”

克拉克不常使用他的超级大脑，此时才反应过来，原来世界上还有一个人没看过那条该死的邮件——即便他的名字也在邮件里。布鲁斯是一个专注又冷静的人，他很可能给自己做了些花边新闻过滤器之类的东西，因此才能在现代社会躲开头条热搜的污染。

他最好在自己的交际圈里留下一个没被污染过的脑子。想到这里，克拉克决定，绝对不能让布鲁斯发现那封该死的邮件。所以他立刻道：“哦，哦，你说这条新闻。”

布鲁斯已经听出他的语气不正常：“你没事吧？”

“没什么，没什么……咳，大都会的几个超级反派那里我都看了，但似乎都不是幕后黑手。”

“嗯。”

克拉克：“你有什么想法？”

“这不是超级反派的会做的事，更像恐怖分子。”布鲁斯委婉地批评了他的思路。至少对蝙蝠侠而言，这已经算是很委婉了。然后说出自己的看法：“我觉得沃勒会有情报。但是……今晚我被安排了新闻发布会。”

布鲁斯沉重地叹了口气：“我希望你晚上来冒充我。”

这不是他第一次找克拉克干这个，克拉克通常是不大乐意的，但今天他立刻答应：“好，几点？”

<br>

不出所料，替布鲁斯背完发言稿，进入自由提问环节，他迎来的第一个问题就是：“韦恩先生，请问今天热传的邮件内容是真的吗？你是否持有莱克斯·卢瑟和超人的不雅照片？”

克拉克模仿着布鲁斯一贯的公众形象，茫然地看着他的同行：“这和我刚刚说的东西有关系吗？”

记者总是勇敢的：“请正面回答我的问题。”

从专业的角度上来说，克拉克应当不予理睬。他也确实是这样做的：“还有人有问题吗？”

记者们齐刷刷地举起手。

“与卢瑟或超人无关的？”

记者们齐刷刷地把手放下。
