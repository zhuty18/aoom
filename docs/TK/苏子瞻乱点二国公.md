# 苏子瞻乱点二国公

有温荆。这次玩点小众的，苏轼迫害温荆。

有私设名士。虽然忘川连饼上的芝麻都没给我画过半粒，但是我喜欢他，所以我就要写。

<br>

使君从九泉之井中捞出了新名士。

不是旁人，正是文学家、官员、学者，小贾谊，小苏轼，十四岁名满京城，十八岁会试擢卷，二十四岁状元及第翰林修撰，长安街上的通宵琵琶手，大礼议中的核心辩论员，中国历史上唯一组织朝臣对皇帝示威的政治活动家，左顺门的嚎哭臣，滇南的流亡客，诗词歌赋无一不精，作毛宗岗三国演义开篇，经史子集部部全通，开四百年考据学风滥觞，有明一朝二百七十六年内第一才子，比朱厚熜更合理的皇位继承人，永远年轻，永远骂人难听，永远热血，永远不识好歹——杨慎，杨升庵是也。

<br>

陆游：“哇，状元。”

苏轼：“哇，状元。”

王安石：“哇，状元。”

忘川文人中，因为种种原因与状元失之交臂的有，但要说正经八百琼林落首座、打马游御街的状元郎，那杨慎还真是头一个。不过他并没穿那身随时可以拉去拜堂的大红状元服色，甚至官服都没穿，只是系了单纱半臂，背负琵琶，笑呵呵地道：“你好啊，听弹词吗？”

根据故世惯例，名士来到忘川，往往习惯性地就先去拜见本朝皇帝。忘川风华录对开国之君的偏好极大程度上方便了这一点，即便形影相吊如赵云，也一样能找到刘邦刘彻刘秀拜一拜。

杨慎也不例外——虽然他在故世逮一个皇帝骂一个，但对于太祖太宗，他还是尊重的。所以在正常的拜见流程后，他提议：“要不我给您三位弹一段吧？”

拒绝官员表演，拒绝不了官员硬要表演。朱元璋自负身份不开口，马皇后慈祥地看着他们笑，朱棣眼睛一亮：“好哇！”
