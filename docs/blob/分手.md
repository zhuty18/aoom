# 分手

一个中学生AU



Bruce：我认为我们应该分手

Hal：啥？

Bruce：我是指，公开分手，然后私底下再复合

Hal：为什么？

Bruce：因为自从我承认和你在一起了，所有人看我的眼神都非常奇怪

Hal：相信我，要是和我分手了，他们看你的眼神就会从“天下还有这么好运的人”，变成“世上怎么会有这种瞎子”，只会更奇怪

Bruce：我认为会从“他是不是哪里不太正常”变成“他终于清醒了”

Hal：不，那是看我的眼神

Bruce：呵

Hal：不服是吧，打赌吗？

Bruce：赌什么？

Hal：你要是输了，我们公开复合的时候，你就要承认是你追的我，反之亦然

Bruce：成交
