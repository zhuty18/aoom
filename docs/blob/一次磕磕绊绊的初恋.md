# 一次磕磕绊绊的初恋

聊天记录体

<br>

那谁：你明天有空吗？调查缺个男朋友。

我：没空。

那谁：不可能，我查了你的聊天记录。

我：没兴趣当你的假男朋友。

那谁：就一天。

我：没兴趣。

那谁：我管饭。

我：没兴趣。

那谁：我又没说一定是假男朋友。

我：？

那谁：把我备注改了，你不改我替你改。

Tim：当不当？

我：不。

Tim：为什么？

我：我本来是准备追求你的。

Tim：现在这样多省事

我：我想追求你。

Tim：？？？

Tim：这样吧，你撒个娇，我就让你追求我两星期过把瘾。

我：……撒娇？

Tim：嗯，发条语音，说“哎呀求你了”

我：……

我：[语音]（非常直男还有点不耐烦式的“哎呀求你了”）

我：[语音]（“哎呀，求你啦~”现学现卖的娇俏，非常恶心）

我：你喜欢哪种？

Tim：你……算你狠

Tim：你能不能从后天开始追，我明天需要交往中的男朋友。

我：成
