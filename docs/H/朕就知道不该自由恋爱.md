# 朕就知道不该自由恋爱

很久很久以后，当在酒店大堂里偶遇家长时，嬴政准会想起他第一次见到刘彻的那个晚上。

现代社会有各种各样的好处，但难免有一些弊端。比如现代卓越的工业和基建能让你快速搞定一件礼品，但快递员有时会把长安帝苑西区某栋某单元某户的快递误送去街对面东区的同一门牌号门外。当然，快递员可以明天把它取回来重新投递，但既然这是你计划明天晚上送出的礼物，那你当然会想要亲自跑一趟先把它拿回来看看。

所以十三岁的嬴政走进对街的小区，按响门铃，坐上电梯，来到了陌生的门外。电梯打开，半开的防盗门里，只见一个少年人把另一个拦在臂弯里，拉拉扯扯搂搂抱抱，毫无包袱地撒娇说：“仲卿，你就留下睡呗。”

嬴政头顶的问号冒到一半，那小子已然扫他一眼：“看什么？没见过自由恋爱？”

自由恋爱，嬴政心想，好陌生的词汇。

此时他还不知道说这话的人其实姓刘，不然他绝不会听进去。他只看着刘彻怀里那个人的动作，感到一种微妙的困惑和违和：从肢体语言和微表情上，明显他是不怎么想留下的；但是如果他真的想要拒绝，那钻出来一头扎进电梯跑路却也绝非难事。

那个人指指一侧的鞋柜，对嬴政说：“你的快递。”

嬴政简单道了一声谢，拿上东西走了。回家的路上穿过小区层层叠叠的树影，此时的他还不知道什么叫做以和柔自媚于上，只没来由的想：这就是自由恋爱的样子吗？

他看着手里准备送给家教老师的精装盒子，思忖，朕也想要这种。

翌日，他这辈子第一次见到李斯。

<br>

上周他爸说给他请了个家教，嬴政第一反应其实是冷哼一声——谁家初中生用得上研究生来辅导？什么家教，和之前的管家保姆一样，都只不过是代行父职的一个名头罢了。

然后他爸说：“至少试一次课，你要是能和李斯合得来，我以后也能省点心。”

嬴政本是会拒绝的，但他听到了家教的名字。他陡然想起今年他十三岁，上一个十三岁他坐上了王位，有人辞别了故国、师长和挚友，孤身入秦来到他身边。

李斯。时隔两千多年，嬴政再次允许自己咀嚼起这个名字，会是你吗？

——像。嬴政打量着面前的青年，心想，但他可没你会打扮。

“这是我儿子，小政。小政，这是李老师。”他爸介绍了一下，“你们聊，我去书房了。”

“吕政同学。”李斯扶了一把黑框眼镜，不知是紧张还是什么，还顺手理了下头发。

嬴政淡淡道：“我不姓吕。”

“不好意思……你是随母姓吗？”李斯惭愧一笑。

他牙很漂亮，嬴政心想。所以他继续回答李斯无聊的问题：“不，我母亲身份证上姓赵。——我随我父亲，姓嬴。”

李斯眨眨眼：“那吕先生是……？”

“我爸。”嬴政在脑内检索了一下，剔除掉所有封建余孽的部分，尽量用一种平平无奇的口吻道，“他是我父亲的对象。我父亲身体不太好，大部分时候在国外治病，母亲有她自己的生活，所以我算是我爸养大的。——这些我爸没和你说吗？”

李斯缓缓摇头：“吕先生只告诉我，你，你们家只有你们两口常住。”

嬴政一笑，这避重就轻的商人做派，他爸不愧是领先版本两千年的资本家。

李斯在他桌边坐下：“我本来还在想，你成绩这么好，又不是要学什么课外特长，不知道为什么要请家教。现在看来，吕先生可能只是怕你孤独。”

嬴政不置可否：“李老师，你都教什么？”

“你想学什么？”李斯反问，“大部分东西我都能教，初中课本、国学、音乐……”

“击节长歌”四个字蓦地跳进嬴政脑子里，他花了好大力气才没笑出声来。忍了又忍，终于没问他“越席起舞能教吗”，只是道：“那李老师教教我书法吧。我总觉得写字不太得劲。”

李斯点头，拿了稿纸和中性笔写起来，边写边讲姿势、发力、落笔、字体架构。嬴政等他讲了一段，就拉着李斯的手放到自己手腕上，轻声说：“请老师教我。3”
