# 霍光传里不是这样的

你怎么直接要我做皇帝啊，书里不是这样的。

你应该先和我太爷爷搞好关系，和我直系长辈称兄道弟，干掉其他顾命大臣大权独揽，政自己出，成为事实上的皇帝，天下想闻其风采。再废掉个把不听话的先帝，上表太后把我一顿夸，最后挑一个黄道吉日来让我登基啊。登基之后你还应该给我骖乘，陪我去见列祖列宗，让我如芒在背，历史典故传到两千多年以后去的。

你怎么直接上来就要我做皇帝呀？霍光传里不是这样的！我不接受！

<br>

“……殿下。”于谦不太习惯地叫出这个称呼，规劝道，“国不可无主，太子年幼，请殿下为国家计，勉力为之，万勿推辞。军情如火，耽搁不得。”

朱祁钰不吱声，于谦不明所以，抬眼看他，与年轻的郕王对上目光，只见景泰皇帝盯着他，轻轻一笑。

“大司马请起。”朱祁钰刻意把这称呼念重了些，“小王德薄才浅，非嫡非长，不敢从君所请。说来不怕于卿笑话，早先左顺门的事，如今想来zh……真还令我胆战心惊，惶恐不安，如芒在背，恐无力当天下之任。”

于谦一时不知该气还是该笑：刚说完如芒在背就来一句做不了皇帝，陛下若不是在逗他，那就是他和陛下一起来到了一条宣帝也被废了的神秘世界线。

朱祁钰满脸真诚，装模作样地道：“不瞒大司马，小王只怕今日勉强从君所请，来日愈发身不由己。如今，（他咬了咬牙才挤出这一声）皇兄下落不明，小王不敢言万一，但若天佑，皇兄得归……届时一国二主，卿要我如何自处？”

于谦只好哄他：“受国之垢，是谓社稷主；受国不祥，是为天下王。殿下诚能护国安民，令家国安泰，海晏河清，则天下万民人心自有向背，宁复有他？”

朱祁钰让他吓了一跳，这种话你也敢在宫里说？左右一望，见没外人，才松了口气，含嗔带怒瞪了于谦一眼：你是想劝进还是想造反？
