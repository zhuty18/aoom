# 今天你爱刘邦了吗

仲春时玄鸟飞过 岐山上凤鸣如歌

大泽畔又有人 梦一场遇合

咸阳城繇羡气魄 芒砀山醉斩白蛇

剖鱼腹 学狐说 问谁敢随我

大风起 赤旗扬 据沛县 应陈王

一样崤山函谷 天下兵 聚咸阳

气五色 天子象 冠竹皮 约三章

问周鼎逐秦鹿 换兴亡

风云乱

草木的兵戈不分燕韩
