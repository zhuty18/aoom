# 今天你爱刘邦了吗

仲春时玄鸟飞过 岐山上凤鸣如歌

大泽畔有神蛇 梦一场遇合

曾羡皇帝好气魄 也乘夜色释闾左

空鱼腹 静篝火 诸君谁从我

大风动 赤旗扬 起沛县 应陈王

一样崤山函谷 天下兵 聚咸阳

气五色 天子象 入武关 约三章

问周鼎逐秦鹿 探兴亡

风云乱

草木的锋刃胜过龙泉

不别贵贱 帝子也敢斩

乾坤转

兵燹掠九州直逼星汉

天河烁烁问我 何为平安

<br>

谁随我起青萍末 谁陪我作鸿门客

谁受命高坛上 为我争山河

读遍了前朝书册 挥别了旧日君国

汉中剑 楚地歌 举世尽降我

大风起 云飞扬 布衣身 三尺芒

走过天地壮阔 威海内 归故乡

来听我 歌一场 万万人 同欢唱

立社稷家四海 朝未央

风云散

泗水中草鲤跃鳞在氾

脱去 未脱旧竹冠

乾坤换

从此后人间称名以汉

世世代代一愿 长乐长安

<br>

神仙看

从来兴与衰 都在尘寰

史书上尽是凡人的波澜

君莫叹

两千年故事 仍未讲完

今日芸芸你我 亦书灿烂
