# 芙蓉池夜宴事件

包含要素：

曹丕半夜不睡觉，招呼大家陪他去芙蓉池边飙车

曹植兴致勃勃甚至大加赞誉，把其他人当空气，眼里真就只有他哥

刘桢沦为play一环，看起来在写一种很新的梦游文学，疑似快睡着了

应玚对曹丕疯狂卖惨说你看看我吧我混得太难了，后升为五官中郎将文学

——这不是同人文，他们真干了

这就是建安风骨的集大成事件，三曹七子中至少四位参与（其他人没有诗传下来）的——芙蓉池夜宴事件

看完这篇诗歌鉴赏，你就明白诗才这种东西，三分靠天赋，九十七分靠同行衬托

一、曹丕《芙蓉池作》

乘辇夜行游，逍遥步西园。
双渠相溉灌，嘉木绕通川。
卑枝拂羽盖，修条摩苍天。
惊风扶轮毂，飞鸟翔我前。
丹霞夹明月，华星出云间。
上天垂光采，五色一何鲜。
寿命非松乔，谁能得神仙。
遨游快心意，保己终百年。
