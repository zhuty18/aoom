# 星煌帝国

在半个世纪前，这里是人类文明中最庞大的帝国。领域相当于现在联盟五分之一的领土，也就是三分之一个中央区，或者两个军区加起来。

星煌帝国是人类文明最璀璨的结晶，它有着最前沿的科技，最优秀的教育，最强大的武力，最广袤的疆土。

原本有八万亿人口。

## 难民

[[../年表#星历923年|星煌帝国覆灭]]后，逃到联盟的人，约一百万，如[[顾止]]。

难民与联盟公民结合后的后代，可以加入联盟籍。但如果家长只有星煌难民，那么后裔依然被视作星煌籍的难民，如[[叶修]]。
