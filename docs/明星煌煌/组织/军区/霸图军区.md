# 霸图军区

别名远方军区。边境外的军区。这一代联盟领导人奉行“进攻就是最好的防守”，因此设立了霸图军区，对虫族主动出击，以减轻其他军区的防守压力。

平均实力最强的军区，有自己的内网，通常不上公网。
