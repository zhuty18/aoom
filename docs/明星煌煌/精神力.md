# 精神力

机甲驾驶员所具备的一种场，可以与[[机甲#机甲核|机甲核]]共鸣，从而驾驶[[机甲]]。

## 精神体

精神体的珍稀程度往往可以反应精神力强度。通常标准如下：

日常小动物：一至六星。

稀有动物：七至八星。

传说级动物：九至十星。

```dataview
TABLE WITHOUT ID
file.link as 人物,
vi as 实力,
spiritus as 精神体
WHERE vi
SORT choice(vi="十星",10,choice(vi="九星",9,choice(vi="八星",8,choice(vi="七星",7,choice(vi="六星",6,choice(vi="五星",5,4)))))) DESC
```
