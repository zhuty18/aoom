# 叶修

原[[嘉世军区|嘉世]]司令，[[精神力#精神体|精神体]]青龙，[[一叶之秋]]前任驾驶员，[[联盟]]第一人，联盟四大战术大师之一，联盟唯一的前线元帅。

青龙形态：青鳞金角，蛇身鱼尾，平时黑眼睛，约小臂长短（抻直后有胳膊那么长）。战斗形态眼睛为血红，盘旋态约有两米长。

遭[[嘉王朝]]暗算，命悬一线时被[[陈果]]和[[唐柔]]捡回[[兴欣机甲回收站|兴欣]]。
