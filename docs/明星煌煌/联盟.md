# 联盟

联盟成立于

## 军区

[[联盟]]最高级别的军事实体。

共五个：嘉世军区、霸图军区、微草军区、百花军区、蓝雨军区。中央军因为编制类似，常常被误称作中央军区。

### 嘉世军区

别称东方军区，因司令[[叶修|叶秋]]精神体而得名。

### 蓝雨军区

别名西方军区，因副司令[[黄少天]]精神体而得名。

司令[[喻文州]]，副司令[[黄少天]]。

### 微草军区

别名北方军区，因司令[[王杰希]]、副司令[[方士谦]]精神体合体形态而得名。
